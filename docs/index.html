Project Overview

CodeTrack is an innovative web application designed to automate the tracking of students' programming progress on CodeChef. Built with Flask and modern web technologies, it provides educators with a robust platform to monitor multiple classes and track individual student progress through automated web scraping.

Project Goals

  • Automate the process of tracking student progress in competitive programming
  • Provide real-time insights into student performance
  • Enable efficient management of multiple classes
  • Reduce manual effort in progress tracking

Setup Instructions

Prerequisites

  • Python 3.7 or higher
  • pip package manager
  • Git (for cloning the repository)

Installation Steps

# Clone the repository
git clone https://github.com/yourusername/codetrack.git
cd codetrack

# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Initialize database and run the application
python app.py

Dependencies

  • Flask - Web framework
  • BeautifulSoup4 - Web scraping
  • Requests - HTTP client
  • Pandas - Data processing
  • SQLite3 - Database
